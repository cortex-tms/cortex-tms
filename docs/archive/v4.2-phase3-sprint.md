## Context

Phase 3 extended the preset system from Node-only to a three-ecosystem pack. Node preset was first
hardened (TMS-420), then upgraded with automatic package manager detection (TMS-426). Python and Go
presets were added from scratch (TMS-427, TMS-428). Each preset ships 6 template override files that
replace base template content with ecosystem-idiomatic conventions, patterns, and tooling.

All three presets share the same resolver logic (`resolveTemplateSource`) and are independent of
`--scope` (which controls the file set). No package manager detection was added for Python or Go —
Go uses `go` uniformly; Python PM detection is queued.
