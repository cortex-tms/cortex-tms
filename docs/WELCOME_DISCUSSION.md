## 🚀 What is Cortex TMS?

Cortex TMS is the universal AI-optimized project boilerplate that helps you:
- **Scaffold better projects** with AI-friendly documentation structure
- **Review code with Guardian** using LLM-powered pattern detection
- **Organize files intelligently** with git-based auto-tiering
- **Work faster with AI tools** like Claude Code and GitHub Copilot

**Current stats** (February 2026):
- ⭐ **146 GitHub Stars**
- 📦 **NPM Package**: [cortex-tms](https://www.npmjs.com/package/cortex-tms)
- 🌐 **Website**: [cortex-tms.org](https://cortex-tms.org)
- ✅ **100% test pass rate** (production-ready)
